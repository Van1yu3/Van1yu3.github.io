巴别塔

发表于 2019-12-27

那时，天下人的口音、言语都是一样。他们往东边迁移的时候，在示拿地遇见一片平原，就住在那里。他们彼此商量说：“来吧！我们要做砖，把砖烧透了。”他们就拿砖当石头，又拿石漆当灰泥。他们说：“来吧！我们要建造一座城和一座塔，塔顶通天，为要传扬我们的名，免得我们分散在全地上。”耶和华降临，要看看世人所建造的城和塔。耶和华说：“看哪，他们成为一样的人民，都是一样的言语，如今既做起这事来，以后他们所要做的事就没有不成就的了。我们下去，在那里变乱他们的口音，使他们的言语彼此不通。”于是耶和华使他们从那里分散在全地上；他们就停工，不造那城了。因为耶和华在那里变乱天下人的言语，使众人分散在全地上，所以那城名叫巴别。
——圣经创世纪11章1-9节

我走出开一氧化二氢之门，看到前辈正迎面走来，一脸严肃，没有一丝笑容。前辈的眼睛直视着我，在充满白光的过道里透着清冷的光辉。

我想要友好地笑笑，但前辈的那副表情让我只如同冷笑一般扬起了嘴角，笑容产生的气息没能从嘴里释放，只能通过鼻子冷哼而出。

前辈大跨步从我身边走过，步伐坚定而迅速，全身上下仿佛都透露出对失败者的不屑和嘲弄。

我不禁咀嚼着屈辱的当下：在赛博空间里，原本以为我的征途是星辰大海，我可以劈波斩浪所向披靡，然而万万没想到我在岸上便跌了个跟头，在星辰大海的入海口又被小波小浪玩弄得晕头转向。我在各种尝试中多次溺水，错误的尝试还导致一波未平一波又起，我连连受挫，在苦海里爱恨交织。而我在苦海中伸手求援时，前辈还把我按到水里，多灌了几口苦水。

想到这儿时，我已经又回到赛博空间的入口前。我冷笑一声，既然前辈待我如此，我又何必热脸贴冷屁股，自取其辱！我不会再在前辈面前无所顾忌地分享自己的失败经验和教训！

约摸一万个时间单位之后，我疲惫地退出强者无所不能而弱者垂死挣扎的赛博空间，伤痕累累。

与前辈前去摄入能量的路上，前辈主动开口：“怎么样了？”我沮丧地摇摇头，叹了口气。前辈笑道：“遇到问题多Gg，实在不行找Mc，我也是这样过来的。”

凝聚在我心中的火药味忽然间烟消云散。
